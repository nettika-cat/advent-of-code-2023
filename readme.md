# Advent of Code 2023

My solutions to [Advent of Code 2023](https://adventofcode.com/2023) in Python.

## Setup

Create and activate a virtual environment:

```sh
python -m venv .venv
source .venv/bin/ativate
```

With the virtual environment actiated, install dependencies:

```sh
pip install .
```

## Usage

```sh
python <day #>
```
